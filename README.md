# Product Manager App (Android + Kotlin + Jetpack Compose)

This is a starter project structure for a Product/Task Manager Android app.

It contains only the **project structure and placeholder files** so beginners can start implementing features.

## Features to Implement
- Product list screen
- Add product screen
- Product detail screen
- Navigation using Compose Navigation
- API integration using Retrofit
- Local storage (Room or SharedPreferences)

## Steps
1. Open the project in Android Studio
2. Sync Gradle
3. Implement the logic inside the placeholder Kotlin files
4. Run the app using an emulator

This structure follows **MVVM architecture**.
